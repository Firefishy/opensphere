OGC Servers (like GeoServer) can provide map data via Web Map Service (WMS) or Web Map Tile Service (WMTS) and feature data via Web Feature Service (WFS).

A WMS URL looks like:

https//example.com/ogc?service=WMS&version=1.1.0
or
https://example.com/mapservice/wms?version=1.0.0

Supported WMS versions are: 1.0.0, 1.1.0, and 1.2.0

A WFS URL looks like:

https//example.com/ogc?service=WFS&version=1.1.0
or
https://example.com/mapservice/wfs?version=1.0.0

Supported WFS versions are: 1.0.0, 1.1.0, and 1.2.0

A WMTS URL looks like:

https//example.com/ogc?service=WMTS&version=1.1.0
or
https://example.com/mapservice/wmts?version=1.0.0

Supported WMTS versions are: 1.0.0, 1.1.0, and 1.2.0,
