GeoServers can provide map data via Web Map Service (WMS) and feature data via Web Feature Service (WFS). These services will be automatically detected from the base OWS URL.

Please provide the OWS URL, similar to the format below:
https://example.com/geoserver/ows

To provide specific service URL's or for WMTS support, add the server as an OGC Server instead.

Supported WMS versions: 1.1.1, 1.3.0
Supported WFS versions: 1.1.0
