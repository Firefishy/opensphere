OGC servers can provide map data via Web Map Service (WMS) or Web Map Tile Service (WMTS) and feature data via Web Feature Service (WFS).

A URL for WMS can have many different formats, such as:
  • example.com/ogcserver/wms
  • example.com/ogc?service=WMS&version=1.1.0
  • example.com/mapservice/wms?version=1.0.0
  • example.com/mapservice/wms?version=1.0.0[&parameter=value[â€¦]]
  • example.com/wmsconnector/com.esri.wms.Esrimap
  • example.com/ArcGIS/services/mapservice/MapServer/WMSServer
A URL for WFS and WMTS would look similar to those above, except instances of "WMS" would be replaced by "WFS" or "WMTS" respectively.

Supported WMS versions are: 1.1.1, 1.3.0
Supported WFS versions are: 1.1.0
Supported WMTS versions are: 1.0.0
