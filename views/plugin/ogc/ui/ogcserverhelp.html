Please provide a valid GetCapabilities URL for each supported service on the OGC server.

A GetCapabilities URL can have many different formats, such as:
  • example.com/ogc?service=wms&version=1.3.0&request=GetCapabilities
  • example.com/ogc?service=wfs&version=1.1.0&request=GetCapabilities
  • example.com/ogc/wms?version=1.3.0&request=GetCapabilities
  • example.com/ogc/wfs?version=1.1.0&request=GetCapabilities
  • example.com/geoserver/gwc/service/wmts?REQUEST=GetCapabilities
  • example.com/ArcGIS/services/mapservice/MapServer/WMSServer
Instances of WMS, WFS, and WMTS in the URLs above are often interchangeable.

Supported WMS versions: 1.1.1, 1.3.0
Supported WFS versions: 1.1.0
Supported WMTS versions: 1.0.0
